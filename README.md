# EM Playbook

A personal reference library of prompts, templates, and outputs 
for recurring engineering management situations.

## Structure

| Folder | Covers |
|--------|--------|
| 01-strategic-planning | Org-wide plans, target setting, measurement design |
| 02-stakeholder-communication | Leadership briefs, cross-geo framing, difficult conversations |
| 03-team-management | Manager diagnostics, feedback, recognition, team health |
| 04-technical-architecture | Build vs buy, CI/CD design, metric systems, TDRs |
| _meta | How to use this repo, prompt writing guide, LLM evaluation |

## How to Use

Each situation folder contains three things:

- **prompts/** — context prompts to paste into an LLM to start 
  or continue work on that situation type
- **templates/** — reusable document structures for outputs
- **outputs/** — dated, anonymised outputs from real situations

## Conventions

- All outputs are anonymised — no org names, no individual names, 
  no project names, no internal data
- Prompts include a master context section so any LLM can 
  pick up where you left off
- Dated output folders use YYYY-situation-name format
- Templates are starting points — edit before use
